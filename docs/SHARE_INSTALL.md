# Install ELSD on a phone (friend / family sideload)

**No Android Studio required.** You only need the APK file and a few taps.

ELSD is an **open-source camera visual mixer** (Praxis Software · MIT).  
It uses the **camera** (and optionally the **microphone** for voice). Nothing is uploaded unless you later use a feature that explicitly says so — **1.0 stays on the device**.

---

## Get the APK

1. Open the latest **GitHub Release** for this project:  
   https://github.com/jasonzacherychristie-dev/ELSD/releases  
2. Download **`ELSD-0.1.0-m1-sideload.apk`** (or the newest `ELSD-*-sideload.apk`).  
3. Send it to the phone (link, Drive, AirDrop-style, USB, chat).

Or build it yourself: see [`DEV_DEPLOY.md`](DEV_DEPLOY.md).

---

## Install on the phone (Android)

### One-time: allow installs

**Pixel / stock Android**

1. Open the APK from Files / Downloads / Chrome.  
2. If prompted: **Settings → allow from this source** → turn **on** → back → install.

**OnePlus / OxygenOS**

1. Open the APK.  
2. Allow **Install unknown apps** for that app (Files / Chrome).  
3. Install.

### Install

1. Tap **Install**.  
2. Open **ELSD**.  
3. When asked, allow **Camera** (required).  
4. Allow **Microphone** if you want voice commands (optional for looking only).

### First run

1. You’ll see the **SWITCHBOARD**.  
2. Tap **GO LIVE** for the camera mix (landscape).  
3. Try **ADD EFFECT** → e.g. **SOLARIZE**, **NOIR**, or **RANDOM**.  
4. Say **clear** or use CLEAR if things get too weird.

## Tips

| | |
|--|--|
| **Best phones** | OnePlus 9 / 12 class; any modern Android 8+ with a camera |
| **Orientation** | Live view prefers **landscape** |
| **Exit weird** | CLEAR / sober on the board, or force-close the app |
| **Update** | Download a newer release APK and install over the old one |
| **Uninstall** | Long-press app icon → Uninstall |

---

## Safety (quick)

- Flashing / heavy FX exist — take breaks; don’t drive or walk busy streets with wet effects.  
- **CLEAR** is always available on the switchboard.  
- This is experimental public alpha, not a medical or safety device.
